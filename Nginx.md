# Nginx

--------------------------------------------------------------------------------

## 安装

```shell
brew install nginx
```

## 启动

```shell
sudo nginx #打开
sudo nginx -s reload|reopen|stop|quit  #重新加载配置|重启|停止|退出
sudo nginx -t   #测试配置是否有语法错误
```

## 测试

```shell
http://localhost:8080
```

## 配置

```shell
/usr/local/etc/nginx/nginx.conf
```
